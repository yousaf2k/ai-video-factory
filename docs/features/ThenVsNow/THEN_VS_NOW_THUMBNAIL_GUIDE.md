# ThenVsNow Thumbnail Prompt Guide

**Date:** March 12, 2026
**Status:** Updated

---

## Overview

This guide provides comprehensive specifications for generating YouTube thumbnails for "Then vs Now" reunion videos. Thumbnails feature the most prominent character from the movie in a close-up selfie composition showing both their THEN (younger) and NOW (older) versions together.

---

## Thumbnail Composition

### Subject Selection

**CRITICAL:** Feature the MOST PROMINENT/ICONIC character from the movie

**Examples:**
- The Godfather → Marlon Brando as Don Vito Corleone
- Titanic → Leonardo DiCaprio as Jack Dawson
- Back to the Future → Michael J. Fox as Marty McFly
- The Matrix → Keanu Reeves as Neo
- Jurassic Park → Sam Neill as Dr. Alan Grant

### Visual Layout

```
┌─────────────────────────────────────┐
│  Production Set Background          │
│  (cameras, lights, rigging)         │
│                                     │
│   [YOUNG]  [OLDER + iPhone]       │
│    THEN          NOW                │
│   (left)       (right)              │
│                                     │
│  Crew members (slightly blurred)    │
└─────────────────────────────────────┘
```

**Elements:**
1. **Left side:** Younger THEN version of character
2. **Right side:** Older NOW version holding iPhone 15 Pro Max
3. **Background:** Production set with visible equipment
4. **Crew:** 2-3 members, slightly blurred
5. **Shot type:** Close-up selfie (not full body)
6. **Expression:** Both characters smiling

---

## Mandatory Elements

### 1️⃣ Character Composition

**Requirements:**
- ✅ **Close-up selfie shot** (waist up or tighter)
- ✅ **Both characters visible** in same frame
- ✅ **Younger THEN on left**, older NOW on right
- ✅ **NOW character holds iPhone 15 Pro Max** for selfie
- ✅ **Both characters smiling** at camera
- ✅ **Fun, nostalgic expression** on older character

**Composition:**
```
Close-up selfie of [Character] from [Movie] — Then + Now,
smiling and taking a selfie together on the filming set
```

### 2️⃣ iPhone 15 Pro Max Details

**CRITICAL:** Phone must be clearly visible

**Required Description:**
```
iPhone 15 Pro Max with titanium frame and triple camera system
clearly visible in actor's hand
```

**Additional Details:**
- Phone screen showing camera interface
- Natural selfie arm position
- Phone clearly visible in camera frame
- Authentic Apple product design

### 3️⃣ Production Set Background

**Required Equipment:**
- ✅ Film cameras on dollies
- ✅ Lighting rigs (HMI lights, softboxes)
- ✅ Scaffolding or grip equipment
- ✅ C-stands with diffusion flags
- ✅ Crane arms or overhead rigging

**Set Pieces:**
- Iconic props from the movie
- Recognizable set architecture
- Period-accurate furniture/decor

### 4️⃣ Production Crew

**Requirements:**
- ✅ 2-3 crew members in background
- ✅ Naturally moving, not posing
- ✅ Slightly blurred (depth of field)
- ✅ Authentic crew roles:
  - Camera operator
  - Gaffer adjusting lights
  - Grip moving equipment

**Why:** Creates authentic behind-the-scenes atmosphere without distraction

### 5️⃣ Style & Quality

**Keywords:**
```
Cinematic
Photorealistic
Ultra-realistic textures
Vibrant colors
High-quality YouTube thumbnail style
Natural depth-of-field
Soft cinematic lighting
Hyper-realistic skin
Fun and nostalgic atmosphere
```

## Format Specifications

### 16:9 Thumbnail (Horizontal)

**Usage:** Standard YouTube thumbnail

**Template:**
```
Close-up selfie of [Most Prominent Character] from [Movie Title (Year)] —
Then + Now, smiling and taking a selfie together on the filming set. NOW
version holding smartphone (iPhone 15 Pro Max with titanium frame and triple
camera system clearly visible) for the selfie. Both characters visible: younger
THEN version on left, older NOW version on right holding phone. Visible cameras,
lighting rigs, scaffolding in background. A few production crew members moving
naturally, slightly blurred. Cinematic, photorealistic, ultra-realistic textures,
vibrant colors, high-quality YouTube thumbnail style. Natural depth-of-field
emphasizing THEN and NOW, soft cinematic lighting, hyper-realistic skin, fun
and nostalgic atmosphere. 16:9 horizontal composition.
```

### 9:16 Thumbnail (Vertical)

**Usage:** YouTube Shorts, TikTok, Instagram Reels

**Template:**
```
Vertical close-up selfie of [Most Prominent Character] from [Movie Title
(Year)] — Then + Now, smiling and taking a selfie together on the filming
set. NOW version holding iPhone 15 Pro Max visible in frame for selfie.
Younger THEN version visible beside older NOW version. Production equipment
(cameras, lights, rigging) and crew members in background, slightly blurred.
Cinematic, photorealistic, ultra-realistic textures, vibrant colors,
high-quality YouTube thumbnail style. Natural depth-of-field emphasizing both
characters, soft cinematic lighting, hyper-realistic skin, fun and nostalgic
atmosphere. 9:16 vertical composition.
```

## Common Mistakes to Avoid

### ❌ Wrong Character
**Bad:** Supporting character or obscure role
**Good:** Most prominent/iconic main character

### ❌ Full Body Shot
**Bad:** Full body composition makes characters too small
**Good:** Close-up selfie for maximum impact

### ❌ Missing iPhone
**Bad:** Phone not visible or generic "smartphone"
**Good:** iPhone 15 Pro Max with titanium frame and triple camera system

### ❌ Clean Background
**Bad:** Plain background or solid color
**Good:** Production set with visible equipment and crew

### ❌ Dull Colors
**Bad:** Muted or desaturated colors
**Good:** Vibrant, eye-catching colors

### ❌ No Production Feel
**Bad:** Studio portrait look
**Good:** Behind-the-scenes authenticity with equipment and crew

## Technical Specifications

### Resolution
- **16:9:** 1280x720 minimum, 1920x1080 recommended
- **9:16:** 720x1280 minimum, 1080x1920 recommended

### File Format
- PNG for highest quality
- JPG if file size is concern

### Color Space
- sRGB for YouTube compatibility
- Vibrant, saturated colors for click-through

### Text Overlays
- Avoid text on thumbnail (YouTube adds title overlay)
- Let image speak for itself

## Usage

### In Story Generation

When the ThenVsNow agent generates a story, it will automatically create thumbnail prompts following these guidelines:

```json
{
  "thumbnail_prompt_16_9": "Close-up selfie of [Most Prominent Character]...",
  "thumbnail_prompt_9_16": "Vertical close-up selfie of [Most Prominent Character]..."
}
```

### For Custom Thumbnails

Users can manually edit thumbnail prompts in the story.json file before generation:

1. Navigate to project folder
2. Open `story.json`
3. Edit `thumbnail_prompt_16_9` and/or `thumbnail_prompt_9_16`
4. Save and regenerate thumbnails
