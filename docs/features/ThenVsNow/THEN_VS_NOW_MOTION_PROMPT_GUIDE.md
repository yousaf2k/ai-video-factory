# ThenVsNow Animation / Motion Prompt Guide

**Date:** March 12, 2026
**Status:** Updated

---

## Overview

This guide provides comprehensive specifications for generating video animations (motion prompts) for "Then vs Now" reunion videos. Each character has two video segments with distinct animation patterns: **Meeting** (THEN MEETS NOW) and **Departure** (TRANSITION).

---

## Video Animation Structure

### Two Segments Per Character

Each character featured in the video has exactly **two video segments**:

1. **Meeting Video** ("THEN MEETS NOW") - Arrival and emotional reunion moment
2. **Departure Video** ("TRANSITION") - Exit and transition moment

**Duration:** Each segment typically 15-30 seconds

---

## 1. MEETING VIDEO: "THEN MEETS NOW"

### Purpose

Show the emotional reunion when the older character meets their younger self on the movie set. This moment captures the wonder, nostalgia, and joy of returning to the iconic location.

### Animation Pattern

```
START: Younger character (THEN) on left, older character (NOW) on right
       ↓
APPROACH: Older character walks toward younger
         ↓
RECOGNITION: Younger character smiles warmly first
            ↓
EMBRACE: Both characters step forward and share a warm heartfelt hug
         ↓
HOLD: Brief moment of emotional connection
      ↓
END: Characters separate, ready to explore set together
```

### Required Elements

| Element | Description |
|---------|-------------|
| **Initial Position** | THEN on left side of frame, NOW on right side |
| **Approach** | Older character walks toward younger |
| **Recognition** | Younger character smiles warmly first |
| **Embrace** | Both characters share a heartfelt hug |
| **Camera** | Side angle or front view capturing the reunion |
| **Background** | Production set visible with crew movement |
| **Lighting** | Cinematic, warm, emotional |
| **Atmosphere** | Nostalgic, emotional connection |

### Motion Prompt Template

```
THEN MEETS NOW: The younger character (THEN) on the left smiles warmly as the
older character (NOW) on the right approaches, the younger version extends arms
for embrace, both characters step forward and share a warm heartfelt hug. Camera
captures the emotional reunion moment from side angle. Production set visible
in background with subtle crew movement. Cinematic lighting, emotional connection,
nostalgic atmosphere, smooth natural animation.
```

### Scene Action Template

```
[Meeting/THEN MEETS NOW] Younger character (THEN) on left side of frame smiles
warmly as older character (NOW) on right approaches. Younger version extends arms,
both characters step toward each other and share a warm heartfelt hug. Camera
captures the emotional reunion moment. STRAIGHT-ON EYE-LEVEL CAMERA, PERFECTLY
CENTERED SYMMETRICAL FRAMING, TRIPOD PERSPECTIVE. Production set visible in
background with crew members moving naturally.
```

## 2. DEPARTURE VIDEO: "TRANSITION"

### Purpose

Show both characters leaving the set together, symbolizing their continued connection and friendship transcending time. This moment provides closure and a sense of ongoing relationship.

### Animation Pattern

```
START: Both characters standing together after the meeting
       ↓
MOVEMENT: They walk together to the right side of frame
         ↓
INTERACTION: Smiling and talking in friendly conversation
            ↓
CAMERA: Smooth tracking shot following their movement
        ↓
EXIT: They walk out of scene together
      ↓
FADE: Scene fades to black or cross-fades to next location
```

### Required Elements

| Element | Description |
|---------|-------------|
| **Starting Position** | Both characters together in frame |
| **Movement** | Walk together to the right side |
| **Conversation** | Smiling and talking naturally |
| **Camera Tracking** | Smooth follow shot |
| **Background** | Production set visible as they exit |
| **Crew** | Natural crew movement in background |
| **Lighting** | Warm, hopeful, closure |
| **Atmosphere** | Joyful, friendly, continued friendship |

### Motion Prompt Template

```
TRANSITION: Both characters (THEN and NOW versions) walk together to the right
side of frame, smiling and talking to each other in friendly conversation. Camera
follows their movement with smooth tracking shot as they exit the scene. Production
set and equipment visible in background, crew members working naturally. Warm
lighting, joyful atmosphere, natural walking animation, sense of closure and
continued friendship.
```

### Scene Action Template

```
[Departure/TRANSITION] Both characters (THEN on left, NOW on right) walk together
to the right side of frame, smiling and talking to each other in friendly
conversation. Camera follows their movement with smooth tracking as they exit the
scene together. STRAIGHT-ON EYE-LEVEL CAMERA, PERFECTLY CENTERED SYMMETRICAL
FRAMING, TRIPOD PERSPECTIVE. Production set and equipment visible, crew members
working naturally in background.
```

## Technical Specifications

### FLFI2V Workflow

**Workflow Configuration:**
```python
# In config.py
VIDEO_WORKFLOWS = {
    "wan22_flfi2v": {
        "workflow_path": "...Wan22_FLFI2V_FixSlowMotion_API.json",
        "load_image_first_node_id": "128",  # THEN image (first frame)
        "load_image_last_node_id": "151",   # NOW image (last frame)
        "motion_prompt_node_id": "93",      # Motion prompt injection
        "wan_video_node_id": "150",         # Video output
    }
}
```

**Input Images:**
- **First Frame:** THEN image (younger character alone on set)
- **Last Frame:** NOW image (both characters with selfie composition)

**Motion Prompt:** Meeting or Departure prompt from this guide

### Camera Movement

| Video Type | Camera Movement |
|------------|----------------|
| **Meeting** | Subtle push-in or static side angle |
| **Departure** | Smooth tracking following movement to right |

### Animation Style

**Character Movement:**
- Natural human motion
- Smooth transitions between poses
- Realistic walking animation
- Natural facial expressions
- Emotional authenticity

**Quality:**
- Ultra-realistic motion
- Consistent frame rate
- No jerky movements
- Natural physics
- Human-like timing

### Duration Planning

**Target Durations:**
- **Per segment:** 15-30 seconds
- **Tolerance:** ±5 seconds
- **Total video:** Sum of all scene durations = target video length

**Example for 60-second video:**
- 4 characters × 2 segments each = 8 segments
- 60 seconds ÷ 8 segments = 7.5 seconds per segment
- *Too short!* Reduce characters or increase video length

**Better approach:**
- 3 characters × 2 segments each = 6 segments
- 60 seconds ÷ 6 segments = 10 seconds per segment
- *Acceptable for shorter videos*

**Ideal for 60-second video:**
- 2 characters × 2 segments each = 4 segments
- 60 seconds ÷ 4 segments = 15 seconds per segment
- *Perfect for emotional storytelling*

## Why These Animations Work

### Meeting (THEN MEETS NOW)

**Emotional Impact:**
1. **Recognition:** Younger character seeing their older self creates immediate emotional resonance
2. **Reunion:** Hug is a universal gesture of welcome and love
3. **Wonder:** Captures the magical moment of meeting oneself
4. **Nostalgia:** Viewers feel nostalgia seeing both versions together
5. **Cinematic:** Hug is visually engaging and emotionally powerful

**Visual Storytelling:**
- Shows passage of time directly
- Creates empathy with both versions
- Establishes emotional connection
- Sets up exploration of iconic set
- Hooks viewer emotionally

### Departure (TRANSITION)

**Closure and Continuity:**
1. **Completion:** Walking away together signifies scene completion
2. **Friendship:** Smiling conversation shows ongoing bond
3. **Hope:** Positive tone suggests美好的未来
4. **Movement:** Walking creates natural, dynamic exit
5. **Tracking:** Camera following creates cinematic quality

**Narrative Function:**
- Provides satisfying ending to each character segment
- Suggests characters remain connected beyond the video
- Creates smooth transition between scenes
- Maintains emotional engagement
- Sets up next character introduction

## Common Mistakes to Avoid

### ❌ Meeting Mistakes

**Wrong:**
```
Older character walks in, younger stands still, they nod at each other
```

**Right:**
```
Younger character on left smiles warmly as older approaches,
both step forward and share a warm heartfelt hug
```

**Why:** Hug creates emotional connection; nod is too distant

---

**Wrong:**
```
Characters from different movies meeting
```

**Right:**
```
Younger and older versions of SAME character meeting
```

**Why:** THEN/NOW concept requires same person at different ages

---

**Wrong:**
```
Meeting happens in empty void
```

**Right:**
```
Meeting happens on iconic movie set with production equipment
```

**Why:** Production set authenticity is core to the concept

---

### ❌ Departure Mistakes

**Wrong:**
```
Characters walk away from each other in opposite directions
```

**Right:**
```
Both characters walk together to the right as they exit
```

**Why:** Walking together symbolizes continued connection

---

**Wrong:**
```
Characters walk silently with no interaction
```

**Right:**
```
Characters walk together, smiling and talking in friendly conversation
```

**Why:** Conversation shows ongoing friendship and bond

---

**Wrong:**
```
Camera stays static while characters exit
```

**Right:**
```
Camera follows their movement with smooth tracking shot
```

**Why:** Tracking creates dynamic, cinematic exit

## Usage

### In Story Generation

When the ThenVsNow agent generates a story, it automatically creates motion prompts:

```json
{
  "characters": [
    {
      "name": "Actor Name as Character Name",
      "meeting_prompt": "THEN MEETS NOW: The younger character...",
      "departure_prompt": "TRANSITION: Both characters walk..."
    }
  ]
}
```

### For Custom Prompts

Users can manually edit motion prompts in shots.json before generation:

1. Navigate to session folder
2. Open `shots.json`
3. Edit `meeting_video_prompt` and `departure_video_prompt`
4. Save and regenerate videos
